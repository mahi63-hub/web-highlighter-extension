#  Web Highlighter & Notes – Chrome Extension

A Chrome extension that allows users to **highlight text** on any webpage and **add notes** which persist even after reloading or restarting the browser.


## Features

- Highlight selected text directly
- Highlights automatically saved per webpage URL
- Highlights restored when page is refreshed or browser is reopened
- Popup UI shows list of highlights for current tab
- Click an item in popup → Scrolls to that highlighted part of the page
- Notes can be added and saved for each highlight
